# Installation Steps

### Install git client

1. apt-get install git

Dependencies:
- dos2unix, zip/unzip, tree, openssl
- all packets and distro up-to-date 

### Clone the PKIT master branch
2. clone the master
```shell
git clone https://github.com/KlausWolfinger/PKIT.git
cd PKIT
```
### Set x flag
5. chmod u+x pki.sh

### Launch the PKIT
6. ./pki.sh
